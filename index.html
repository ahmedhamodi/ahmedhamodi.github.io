Mood Detection Box

Developer

From October to December 2017, I worked on a Mood Detection Box that essentially parses tweets for sentiment analysis, and displays a RGB value on an Arduino. I worked on:

  • Used Twitter's API to check trending tweets around the world, filter out non-ASCII character tweets, and then parse all the tweets with those hashtags present.
  • Used IBM Watson's API to perform sentiment analysis on the parsed tweet contents to determine a certainity percentage of various moods.
  • The moods were displayed using an RGB light of an Arduino. A pulsating effect was shown if the secondary mood was close in value, while a solid light was shown if the primary mood was very dominant.
